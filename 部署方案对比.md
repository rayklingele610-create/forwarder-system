# 货代系统 - 国内免费公网部署方案对比

## ✅ 方案一：Serveo.net（推荐，最简单）
**特点**：SSH反向隧道，无需注册，一条命令
**步骤**：
1. 确保 Windows SSH 客户端已启用（已确认）
2. 双击 `serveo_start.bat`
3. 等待10-20秒，会显示公网地址，如 `https://xxx.serveo.net`
4. 用这个地址即可在任何地方访问

**优点**：
- 完全免费
- 无需注册
- 国内访问速度尚可
- 自动HTTPS

**缺点**：
- 每次启动地址可能变化
- 需要保持SSH连接不断开

## ✅ 方案二：ngrok（稳定，需注册）
**特点**：全球知名，稳定可靠
**步骤**：
1. 访问 https://ngrok.com 注册免费账号
2. 下载 ngrok.exe 放到项目目录
3. 获取 token，运行：`ngrok authtoken 你的token`
4. 运行：`ngrok http 5000`
5. 获得 `https://xxx.ngrok.io` 地址

**优点**：
- 非常稳定
- 可自定义子域名（付费）
- 有Web管理界面

**缺点**：
- 免费版每次重启地址变化
- 需要注册

## ✅ 方案三：localtunnel（轻量）
**特点**：Node.js工具，简单快速
**步骤**：
1. 安装Node.js
2. 运行：`npm install -g localtunnel`
3. 运行：`lt --port 5000`
4. 获得 `https://xxx.loca.lt` 地址

**优点**：
- 安装简单
- 地址相对稳定

**缺点**：
- 需要Node.js环境

## ✅ 方案四：Cloudflare Tunnel（最稳定）
**特点**：Cloudflare出品，企业级
**步骤**：
1. 注册 Cloudflare 账号
2. 安装 `cloudflared`
3. 运行：`cloudflared tunnel --url http://localhost:5000`
4. 获得 `https://xxx.trycloudflare.com` 地址

**优点**：
- 企业级稳定
- 国内访问快
- 可绑定自定义域名

**缺点**：
- 配置稍复杂
- 需要注册

## 🚀 推荐执行顺序

### 立即尝试：
1. **双击 `serveo_start.bat`** → 获得临时公网地址
2. 测试手机/外网电脑能否访问

### 长期使用：
1. 注册 **ngrok** 免费账号
2. 下载 ngrok.exe 到项目目录
3. 创建 `ngrok_start.bat`：
   ```batch
   ngrok authtoken YOUR_TOKEN
   ngrok http 5000
   ```

### 最终方案：
如果业务重要，建议：
- 花50元/月租用国内云服务器（阿里云/腾讯云轻量）
- 将整个系统部署到云服务器
- 获得固定IP/域名，24小时稳定运行
